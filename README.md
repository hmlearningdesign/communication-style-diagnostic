# Communication Style Diagnostic

A self-reflection diagnostic tool designed to help professionals identify their communication strengths, recognise development areas, and improve workplace communication effectiveness.

## Project Overview

Communication Style Diagnostic was designed to support self-awareness before scenario-based practice.

Many workplace communication challenges are linked to repeated habits around tone, clarity, confidence, and adaptability. This tool helps learners reflect on their current communication style and understand how their approach may affect collaboration, trust, and performance.

The diagnostic acts as an entry point into a wider communication learning journey, supporting behavioural awareness before learners move into practical application.
